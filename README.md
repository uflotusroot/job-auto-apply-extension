# 自动投递助手浏览器扩展

根据您的技能和心理预期，自动在各大招聘平台投递简历。

## 功能特点

- **智能匹配算法**: 根据技能、薪资、年限、学历等计算职位匹配度
- **多平台支持**: Boss直聘、智联招聘、就业在线
- **自定义筛选**: 关键词筛选、排除关键词、最低匹配度
- **自动翻页**: 自动处理分页，持续投递
- **统计追踪**: 实时显示扫描、匹配、投递数量

## 使用方法

1. 打开招聘平台网站（如 Boss直聘）并登录
2. 点击扩展图标，配置个人信息和筛选条件
3. 点击"保存配置"
4. 进入职位搜索页面，点击"开始自动投递"

## 匹配算法权重

| 维度 | 权重 |
|------|------|
| 技能匹配 | 40分 |
| 薪资匹配 | 20分 |
| 工作年限 | 15分 |
| 学历要求 | 10分 |
| 关键词匹配 | 10分 |
| 地点匹配 | 5分 |

## 注意事项

- 首次使用请先登录招聘平台账号
- 建议设置5-10秒投递间隔，避免操作过于频繁
- 请遵守各平台用户协议
- 扩展不收集任何个人数据，配置仅存储在本地
